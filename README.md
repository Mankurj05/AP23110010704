# Campus Notifications

A small frontend for viewing and prioritizing campus notifications. Includes a local mock backend for testing.

Run locally:

```bash
# start mock backend
node notification_app_be/server.js

# in another terminal, start frontend
cd notification_app_fe
npm install
npm run dev
```

Run tests:

```bash
cd notification_app_fe
npm test
```

Screenshots are in the `screenshots/` folder.
